8. Installation/operation of devices to automatically collecting personal information and refusing this

  • What are Cookies?

    1. The Company uses ‘cookie’ to store and frequently retrieve user information to provide personalized and customized service.

    2. A cookie is a very small text file sent to a browser used by a user through a server used to operate a website and saved at a hard disk of the user’s computer. The website server reads the contents of the cookie stored in the hard disk whenever the user visits the website used to maintain user environments and to provide customized service.

    3. The cookie does not automatically/actively collect information for identifying individuals, and you can refuse to save or delete these cookies at any time.

  • Purpose of using cookies

    1.It helps you to conveniently access and use the website using information contained in cookies when you visit Oh! PCs or a Mobile website. It is also used to provide customized information such as optimized advertisements through records of website visits by users, types of usage.

  • installing/operating and refusing cookies

    1. You have a choice to install cookies. You can allow all cookies or check whenever a cookie is stored, or refuse to store all cookies by setting options from a web browser. Example) For Internet Explorer: Tool menu from the web browser > Internet Option > Personal Information

    2.However, if you refuse to store cookies then you may experience difficulties in using some part of service including areas requiring logins.
